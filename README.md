# Proyecto C
Conversor de Monedas (C)

## Descripción

Esta aplicación es un programa desarrollado en el lenguaje C que permite convertir monedas entre Quetzales (GTQ), Dólares Estadounidenses (USD) y Euros (EUR).
El programa funciona mediante un menú en la terminal, donde el usuario selecciona la conversión que desea realizar e ingresa la cantidad de dinero a convertir.

## Prerrequisitos 
Para ejecutar este proyecto es necesario tener instalado:
Compilador GCC
Terminal o consola
Sistema operativo Linux, Windows o macOS
Instalar GCC en Ubuntu:

sudo apt update
sudo apt install gcc

Verificar instalación:

gcc --version
Pasos para su ejecución

Descargar o clonar el proyecto.

Abrir una terminal en la carpeta del proyecto.

Compilar el programa con el siguiente comando:

gcc conversor.c -o conversor

Ejecutar el programa:

Linux

./conversor

Windows

conversor.exe

Seleccionar la opción de conversión y escribir la cantidad a convertir.*
